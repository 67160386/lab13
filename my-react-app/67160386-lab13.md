# 🎯ส่วนที่ 1: useEffect Basics

## 📋คำถาม 1.1: useEffect dependency array มี 3 ลักษณะ

### A) ไม่ใส่ dependency array

- ทำงานทุกครั้งหลัง render (ทุก render)
- use case: ใช้ในกรณีที่ต้องการ sync อะไรบางอย่างทุกครั้ง เช่น logging

### B) ใส่ empty array []

- ทำงานแค่ครั้งเดียวตอน mount
- use case: fetch API ครั้งแรก, setup event listener

### C) ใส่ตัวแปร [var1, var2]

- ทำงานเมื่อค่าของตัวแปรใน array เปลี่ยน
- use case: fetch ข้อมูลใหม่เมื่อ id เปลี่ยน, search ตาม input

## 📋คำถาม 1.2: Cleanup function ใช้เพื่ออะไร

ใช้สำหรับ "ล้าง effect เก่า" ก่อนที่ effect ใหม่จะทำงาน หรือก่อน component ถูก unmount

**ตัวอย่าง**

1. เคลียร์ interval

```js
useEffect(() => {
  const id = setInterval(() => console.log("tick"), 1000);
  return () => clearInterval(id);
}, []);
```

2. ลบ event listener

```js
useEffect(() => {
  const handler = () => console.log("resize");
  window.addEventListener("resize", handler);
  return () => window.removeEventListener("resize", handler);
}, []);
```

## 📋คำถาม 1.3: Infinite loop

คำตอบ: **B และ C**

**อธิบาย**

- A → ไม่ loop เพราะ run แค่ครั้งเดียว
- B → loop เพราะ setState แล้ว count เปลี่ยน → trigger useEffect ใหม่
- C → loop เพราะ run ทุก render แล้ว setState ทุกครั้ง

# 🎯ส่วนที่ 2: API Fetching

## 📋คำถาม 2.1: Component fetch users

```jsx
import { useEffect, useState } from "react";

function UserList() {
  const [users, setUsers] = useState([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    fetch("https://jsonplaceholder.typicode.com/users")
      .then((res) => res.json())
      .then((data) => {
        setUsers(data.slice(0, 20));
        setLoading(false);
      })
      .catch((err) => {
        setError("เกิดข้อผิดพลาด");
        setLoading(false);
      });
  }, []);

  if (loading) return <p>Loading...</p>;
  if (error) return <p>{error}</p>;

  return (
    <ul>
      {users.map((user) => (
        <li key={user.id}>{user.name}</li>
      ))}
    </ul>
  );
}

export default UserList;
```

## 📋คำถาม 2.2: Race condition คืออะไร

คือการที่ request หลายตัวตอบกลับไม่ตรงลำดับ ทำให้ข้อมูลผิด

**วิธีป้องกัน**

1. ใช้ AbortController ยกเลิก request เก่า
2. ใช้ flag เช่น isMounted หรือ request id เช็คว่า response ล่าสุดเท่านั้นที่ใช้ได้

# 🎯ส่วนที่ 3: Cleanup & Side Effects

## 📋คำถาม 3.1: Timer Component

```jsx
import { useEffect, useState } from "react";

function Timer() {
  const [time, setTime] = useState(0);

  useEffect(() => {
    const id = setInterval(() => {
      setTime((t) => t + 1);
    }, 1000);

    return () => clearInterval(id);
  }, []);

  return <p>{time}</p>;
}

export default Timer;
```

---

## 📋คำถาม 3.2: Event listener และ setInterval ต้อง cleanup ไหม

ต้องทำ cleanup

**เหตุผล**

- ป้องกัน memory leak
- ป้องกัน event ซ้ำ
- ป้องกัน interval ทำงานต่อหลัง component หายไป

# 🎯ส่วนที่ 4: Advanced Patterns

## 📋คำถาม 4.1: AbortController

ใช้สำหรับ "ยกเลิก fetch request"

```js
useEffect(() => {
  const controller = new AbortController();

  fetch("https://jsonplaceholder.typicode.com/users", {
    signal: controller.signal,
  })
    .then((res) => res.json())
    .then((data) => console.log(data))
    .catch((err) => {
      if (err.name !== "AbortError") {
        console.error(err);
      }
    });

  return () => controller.abort();
}, []);
```

## 📋คำถาม 4.2: Custom Hook useFetch

```jsx
import { useEffect, useState } from "react";

function useFetch(url) {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const controller = new AbortController();

    setLoading(true);
    setError(null);

    fetch(url, { signal: controller.signal })
      .then((res) => {
        if (!res.ok) throw new Error("Error");
        return res.json();
      })
      .then((data) => {
        setData(data);
        setLoading(false);
      })
      .catch((err) => {
        if (err.name !== "AbortError") {
          setError(err.message);
          setLoading(false);
        }
      });

    return () => controller.abort();
  }, [url]);

  return { data, loading, error };
}

export default useFetch;
```
